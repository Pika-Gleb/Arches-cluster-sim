# Arches-cluster-sim
modeling the dynamics of the development of the stellar population of the Arches  cluster, taking into account stellar evolution and external factors

Before starting, you need to install all dependencies as shown below:

pip install amuse-framework
pip install amuse-ph4
pip install amuse-seba

pip install numpy
pip install matplotlib
pip install pandas
pip install scipy

pip install pillow
pip install h5py
